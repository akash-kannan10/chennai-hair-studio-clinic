# Chennai Hair Studio Clinic Website

This is a responsive, professional website for **Chennai Hair Studio Clinic**, featuring two branches: **Chennai (Pallikaranai)** and **Trichy (Thillai Nagar)**.

The website is designed to:
- Present clinic services (PRP, GFC, Hair Transplant, etc.)
- Allow patients to book appointments online via Google Calendar
- Showcase before/after results and reviews
- Work well on mobile, tablet, and desktop
